# Vault Nest

Vault Nest is a mobile-first, offline password manager built with Angular and Capacitor. Android records use SQLite; browser records use IndexedDB. Sensitive item payloads are encrypted before either storage engine receives them.

## Install dependencies

Codex intentionally does not install packages in this repository. From WSL2, run:

```bash
npm i @lucide/angular @capacitor/core @capacitor/android @capacitor/camera @capacitor/clipboard @capacitor/filesystem @capacitor/local-notifications @capacitor/splash-screen @capacitor-community/sqlite
npm i -D @capacitor/cli
```

The Android project is generated by GitHub Actions. A local `npx cap sync android` therefore reports that Android has not been added; that is expected unless you first run `npx cap add android`.

## Privacy

Vault Nest does not use Local Storage, Session Storage, cookies, analytics, or remote APIs. Clipboard values copied through the app are overwritten after five minutes while the app process is alive. Android notification copy shortcuts expire after three minutes and are then removed.

## Android CI

CI builds Angular, runs `cap add android`, synchronises Capacitor, and applies the native document-picker, Android Keystore biometric, system-bar, and monochrome notification-icon patches. It then creates signed APK/AAB artifacts when signing secrets are configured.
